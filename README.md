# Importing and updating profiles

## Importing a new profile:

Once you've selected a game, you will see a profile selection page.

To add/import a new profile from a code, click on 'Import/Update'
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/e886116c-d1bb-4cbd-85cd-83c3e13c6223" />


A pop-up will appear on your screen, giving you two options.

Select 'From code'.
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/ff26a449-35bd-4228-bff0-4110455ac6e3" />


A new pop-up will appear, asking you to enter the profile code.

Paste the profile code you want into the text box, then, after making sure it's the correct code, click 'Continue'.
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/c98b050a-b58b-4960-8926-9183698ff29b" />


All of the packages which will be installed from the code will be displayed.

Click 'Import'.
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/35f39da2-512c-47cb-9bc2-192c79e78733" />


It will then ask you if you want to update an existing profile, or create a new one, using the code.

Click 'Import new profile'.
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/0757954a-50f8-43f6-a82f-2bb6865ee0ac" />


It will then ask you to give the new profile a name.

Once you've given the profile a name, click 'Create'.
<img width="1216" height="739" alt="image" src="https://github.com/user-attachments/assets/6654ccf8-f95c-4087-96f2-6e67d517fb0b" />

Downloading and installing the mods should take several minutes.
